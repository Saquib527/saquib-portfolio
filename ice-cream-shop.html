App Description

It is a food ordering system where customer can order Ice Cream and various drinks from a variety of categories of Ice Cream and drinks. User can choose to order items by Cash On Delivery and from their credit or debit cards. User get their Order Update through Push Notification and user can check their order status from within the App. User can add items to cart and to favourite item list for buying them in future. User can also rate items if they like it.

This App has backend of PHP and MYSQL and I used Firebase OTP Authentication for user login with phone and a login form with PHP backend to login with user details. I used Braintree Payment Gateway in this App for payment and Firebase Push Notification for update of order. This App has both client and server version for quick management of Orders.
